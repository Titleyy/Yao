# Motor Control Release Hub for TLE987x

This repository contains the Motor Control Embedded Software Library for TLE987x with quality level **Pre-Alpha / Dev**.

## Content

- Migration of legacy v1.3.0 motor control features to a modularized approach and introduction of platform-independent FocSolution
- Enablement of static configuration through MOTIX™ Solution Designer (MSD) v3.2.1
- Enablement of dynamic parameter handling, including checks, via MSD.
- Support for oscilloscope (included in MSD) through TraceBox hardware and TbStreaming software component
- Exemplary Keil uVision project under ```example_uvision5\```
- Pre-generated binaries in hex- and axf-format with a corresponding pxml for dynamic parameter handling under ```binary\```
